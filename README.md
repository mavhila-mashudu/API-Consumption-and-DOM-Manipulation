# Country Explorer

**Country Explorer** is an interactive web application that allows users to search for countries, view detailed information, and explore neighboring countries. It provides a clean, modern interface with responsive design and smooth user interactions.

## Features

- **Search Countries**: Enter a country name to view detailed information including:
  - Capital city
  - Population
  - Subregion
  - National flag
  - Languages
  - Currencies
  - Country code
- **Bordering Countries**: Displays neighboring countries with their flags.
- **All Countries Button**: Quickly list all available countries for browsing.
- **Responsive Design**: Works well on desktop and mobile devices.
- **Interactive UI**: Hover effects, smooth transitions, and subtle animations enhance the user experience.
- **Error Handling**: Informative messages if the country is not found or there’s a network error.
- **Loading Spinner**: Indicates when data is being fetched.


## Technologies Used

- **HTML5 & CSS3** – Semantic markup, modern styling with gradients, transparency, and transitions.  
- **JavaScript (ES6)** – Fetch API for retrieving country data from [REST Countries API](https://restcountries.com/).  
- **REST Countries API** – Provides country information including borders, flags, population, and regions.
